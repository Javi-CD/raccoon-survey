Introducción
============

**Raccoon Survey UI** es la interfaz gráfica del sistema Raccoon Survey.  
Su propósito es permitir al usuario interactuar con el backend de forma intuitiva, rápida y moderna.

Está desarrollado utilizando tecnologías web estándares:
- **HTML5** para la estructura.
- **CSS3 / TailwindCSS** para los estilos.
- **JavaScript (ES6)** para la lógica y eventos dinámicos.
- **Fetch API** para comunicación con el backend Flask.
