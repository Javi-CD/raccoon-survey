Raccoon Survey Doc
==========================================

Esta documentación ofrece una visión integral del backend de Raccoon Survey. Se genera automáticamente a partir de los Docstrings con estilo Google y de la especificación OpenAPI oficial del proyecto. Aquí encontrarás la referencia de módulos, modelos, servicios, middlewares, rutas y extensiones, junto con la descripción detallada de la API HTTP (endpoints, parámetros, cuerpos de petición y respuesta, códigos de estado y esquemas). El objetivo es facilitar el desarrollo, el mantenimiento y la integración de clientes externos mediante un contrato técnico claro y actualizado.
