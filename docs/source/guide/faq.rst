Frequently Asked Questions (FAQ)
================================

How do I create my first survey?
--------------------------------
Follow “Usage Guide → Create Surveys”. You need creation permissions.

How do I distribute access tokens?
----------------------------------
Generate tokens from the survey and export the file to share. See “Usage Guide → Generate Tokens”.

Can I integrate an external system?
-----------------------------------
Yes. Check “Internal Documentation → OpenAPI” for available endpoints.

Where can I see the results?
----------------------------
Go to “Usage Guide → Reports and Metrics”. You can filter and export.

What should I do with 401/403 errors?
-------------------------------------
See the Security Policy and related documentation: :doc:`Security Policy <../security_policy>` and :doc:`Security Guide <../security_guide>`.
