# Código de Conducta de Raccoon Survey

Este proyecto promueve un entorno abierto, inclusivo y seguro. Todas las personas que participen (usuarios, colaboradores, mantenedores, sponsors) aceptan cumplir este código de conducta en todos los espacios del proyecto.

## Nuestro Compromiso
- Fomentar una comunidad libre de acoso, respetuosa y colaborativa.
- Valorar la diversidad de experiencias y perspectivas.
- Priorizar la seguridad y protección de datos personales.

## Nuestros Estándares
Comportamientos que contribuyen a un ambiente positivo:
- Mostrar empatía y respeto en la comunicación.
- Aceptar críticas constructivas y enfocarse en soluciones.
- Cuidar la privacidad: no compartir ni exponer datos sensibles de encuestas, equipos o personas.
- Cumplir las políticas de seguridad del proyecto ( [SECURITY.md](SECURITY.md)).
Comportamientos inaceptables:
- Acoso, discriminación, lenguaje o conducta ofensiva.
- Divulgación no autorizada de información personal o resultados de encuestas.
- Intentos de acceso no autorizado, abuso de tokens o manipulación de datos.
- Spam, doxxing, amenazas o cualquier forma de violencia.

## Responsabilidades de Cumplimiento
- Los mantenedores y administradores del proyecto deben promover estos estándares, y tomar acciones apropiadas ante incidentes.
- Pueden modificar o rechazar contribuciones que no se alineen con este código.

## Alcance
- Este código aplica en repositorios, issues, pull requests, documentación, foros, canales de comunidad y cualquier interacción relacionada con el proyecto.

## Reporte de Incidentes
- Use los canales definidos en `SECURITY.md` para reportar vulnerabilidades o incidentes.
- Para conducta inapropiada, puede abrir un issue etiquetado (p. ej. `conducta`) o utilizar el canal privado indicado en `SECURITY.md`.
- Incluya detalles suficientes (fecha, lugar, enlaces, capturas) preservando la privacidad de las personas.

## Proceso de Aplicación
- Los reportes serán revisados con prontitud y confidencialidad.
- Se comunicará la resolución y acciones tomadas cuando corresponda.
- Si es necesario, se involucrarán responsables del proyecto y, de ser el caso, plataformas de hosting.

## Consecuencias
- Advertencias formales y petición de cese de conducta.
- Rechazo de contribuciones y restricción temporal o permanente de participación.
- Escalamiento a instancias superiores cuando se infrinjan leyes o políticas.

## Atribución
- Basado en el [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), adaptado a las necesidades de Raccoon Survey.

Gracias por contribuir a una comunidad segura y colaborativa.
